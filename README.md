# fibermodes
Multilayers fiber mode solver

[![Build Status](https://travis-ci.org/cbrunet/fibermodes.svg?branch=master)](https://travis-ci.org/cbrunet/fibermodes)

API documentation available on http://fibermodes.rtfd.org/


Installation
============

Requirements:

- Python >= 3.4
- numpy
- scipy

 For GUI:

 - pyside
 - pyqtgraph
